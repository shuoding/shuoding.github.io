Research Interests and General Thoughts

  • Type Theory and Type Systems
  • Static Program Analysis
  • Logic and Verification
  • Semantics
  • Foundations of Mathematics
  • I believe rigorous static checking techniques, such as advanced type systems and static program analysis, are really effective in producing reliable programs. We still need to rely on the correctness of the specifications and those checking techniques, though.
  • I'm also interested in the foundations of mathematics, such as topics in set theory, classical logic and constructive logic, etc. I think formal methods might be widely used in the verification of mathematical proofs in the future.
  • Industrial influence is also important, although PL research might affect the industry 10 or more years from now.
  • I enjoy working with the Coq proof assistant, and I particularly like writing code in strongly typed functional programming languages like Haskell.
  • I will also keep being open-minded, which means that I'm always interested in trying new things.
