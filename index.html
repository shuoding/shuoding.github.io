Research Interests: Type Theory and Type Systems, Static Program Analysis, Logic and Verification

  • I believe sound, precise, and efficient static checking techniques (type systems, static program analysis, etc.) are effective in producing reliable programs. We still need to trust the specifications and the static checking techniques, though.
  • I'm also intersted in the foundations of mathematics, such as set theory, classical logic and constructive logic, etc. I think formal methods might be widely used in the verification of mathematical proofs in the future.
  • Industrial influence is also important, although PL research might affect the industry 10 or more years from now.
  • I enjoy working with the Coq proof assistant and strongly typed functional programming languages like Haskell.
  • I will also keep being open-minded, which means that I'm always interested in trying new things.
